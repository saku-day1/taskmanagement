## ファイル構成

```
TaskManagement/
├── frontend/               # React + TypeScript フロントエンド
│   ├── src/
│   │   ├── components/     # UIコンポーネント
│   │   ├── App.tsx         # ルートコンポーネント
│   │   └── main.tsx        # エントリーポイント
│   ├── index.html
│   └── tsconfig.json
├── backend/                # Spring Boot バックエンド
│   ├── src/main/java/
│   │   └── com/example/taskmanagement/
│   │       ├── controller/ # REST APIコントローラー
│   │       ├── service/    # ビジネスロジック
│   │       ├── repository/ # DBアクセス層
│   │       └── model/      # エンティティ・DTO
│   └── src/main/resources/
│       └── application.properties
├── docs/                   # ドキュメント群
└── 要件定義書.md
```

## 使用技術

| 技術 | 用途 |
|------|------|
| React + TypeScript | フロントエンドUI構築 |
| CSS | デザイン・レイアウト |
| Java / Spring Boot | REST API提供・ビジネスロジック |
| Spring Data JPA | DBアクセス（ORM） |
| PostgreSQL | タスクデータの永続化 |

### 主要ライブラリ・ツール

| ツール | 用途 |
|--------|------|
| Gradle | Javaビルドツール |
| Vite | フロントエンドビルドツール |
| Axios | フロントエンドからのAPIリクエスト（REST API） |

## 対象外（スコープ外）

以下の機能は今回の開発対象外とする。

- ユーザーログイン・アカウント管理
- 複数ユーザーでの共同利用
- サーバーへのデータ保存
- ドラッグ＆ドロップによるカード移動
- タスクの担当者割り当て
- タスクの検索・フィルタリング機能
