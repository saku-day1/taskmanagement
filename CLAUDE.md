## Git 運用ルール

- ブランチ: `main` のみ（現時点）
- コミットメッセージ: 日本語
- プッシュ先: `origin/main`
- 危険な操作（rm / git reset --hard / git clean 等）は必ず実行前に確認を取る

## このユーザーとの協働ルール

- **言語**: 日本語で回答する
- **コードコメント**: 原則なし（必要な場合のみ最小限）
- **説明スタイル**: 簡潔に。長い説明よりも短い要点を好む
- **作業確認**: 不可逆な操作（削除・強制プッシュ等）は必ずユーザーに確認してから実行する
- **ドキュメント**: 要件定義書.md をハブとして、詳細は docs/ 配下の各ファイルを参照・更新する
